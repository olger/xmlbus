Web Services Addressing 1.0 - Observer

The processor.xsl stylesheet may be used to check a captured log of WS-Addressing SOAP messages against the assertions in the test cases document.

You should use an XSLT processor which supports XPath 2.0 expressions, such as Saxon 8.6.

How this works: mkschema.xsl uses the testcases document to generate a Schematron schema schema.sch which is used by the stylesheet schematron.xsl to generate processor.xsl.

The processor-munger.xslt generates the processor-xmlout.xsl which produces the results as XML - see output.xml.
